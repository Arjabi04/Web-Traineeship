Assignment 2

Solve the following requirements using:
  1. Class based approach
  2. Functional based approach
const user1 = { name: "Harry", age: 24, address: "Jhamsikhel", city: "Lalitpur" };
const user2 = { name: "Ram", address: "Boudha", city: "Kathmandu" };

Requirements:
  • Create a function/method to display user information
  • First show a countdown from 10 to 0 in the console (Should be dynamic, if no parameter is passed fallbacks to 1 second interval)
  • After the countdown ends, console user1's information
  • After 5 seconds, show user2's information
  • You must use the ES6 features: Template Literals, Object Destructuring, Default Parameters, Rest and Spread Operators
File structure:
  • Create a file named class-based.js for the Class approach
  • Create a file named functional-based.js for the Functional approach
  • Create a main app.js file
  • Import both class-based.js and functional-based.js into app.js
